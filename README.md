## Project Setup

Setup development:

```bash
npm ci
npm run dev
```

Open [http://localhost:3000](http://localhost:3000)

Optimized run:

```bash
npm run build
npm run start
```

Open [http://localhost:3000](http://localhost:3000)

### With Docker

Build the images:

```bash
docker compose up
```

The app is now served at [http://localhost:8080](http://localhost:8080)
